Detailed Description

Class that gives ids to inherenced units We differentiate inherenced units on two types. The first one is that we want to add to our database in any case. The second one where we first check, if opponent already has this kind of unit and then we decied to add it or not. This units are differentiate by their IDS. All inherenced units has negative ID values but Units of first type has odd negative IDS and units of second type has even negative IDS.
